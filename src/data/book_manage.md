---                                                                                                           
BookController 接口文档

基础路径：/book，所有接口均为 POST，请求体为 application/json。
                                                                                                                                                                                                   
---                                                                                                                                                                                              
1. 添加书本

POST /book/bookAdd

┌─────────────┬────────┬──────┬─────────────┐                                                                                                                                                    
│    字段     │  类型  │ 必填 │    说明     │                                                                                                                                                    
├─────────────┼────────┼──────┼─────────────┤                                                                                                                                                    
│ usr_id      │ String │ 是   │ 用户ID      │
├─────────────┼────────┼──────┼─────────────┤                                                                                                                                                    
│ book_name   │ String │ 是   │ 书名        │                                                                                                                                                    
├─────────────┼────────┼──────┼─────────────┤                                                                                                                                                    
│ author      │ String │ 是   │ 作者        │                                                                                                                                                    
├─────────────┼────────┼──────┼─────────────┤                                                                                                                                                    
│ description │ String │ 是   │ 书本描述    │
├─────────────┼────────┼──────┼─────────────┤                                                                                                                                                    
│ cover_image │ String │ 是   │ 封面图片URL │
└─────────────┴────────┴──────┴─────────────┘

响应示例：                                                                                                                                                                                       
{               
"success": true,                                                                                                                                                                               
"book_id": "123456_1713000000000",
"message": "Book added successfully"                                                                                                                                                           
}
                                                                                                                                                                                                   
---                                                                                                                                                                                              
2. 删除书本

POST /book/bookDelete

┌─────────┬────────┬──────┬────────────────────┐                                                                                                                                                 
│  字段   │  类型  │ 必填 │        说明        │                                                                                                                                                 
├─────────┼────────┼──────┼────────────────────┤                                                                                                                                                 
│ book_id │ String │ 是   │ 书本ID             │
├─────────┼────────┼──────┼────────────────────┤                                                                                                                                                 
│ usr_id  │ String │ 是   │ 用户ID（权限校验） │                                                                                                                                                 
└─────────┴────────┴──────┴────────────────────┘

响应示例：                                                                                                                                                                                       
{               
"success": true,                                                                                                                                                                               
"message": "Book deleted successfully"
}

▎ 非本人书本返回 {"success": false, "message": "Permission denied"}

┌─────────┬────────┬──────┬────────────────────────────────┐

├─────────┼────────┼──────┼────────────────────────────────┤                                                                                                                                     
│ book_id │ String │ 是   │ 书本ID（作为RAG搜索的subject） │
├─────────┼────────┼──────┼────────────────────────────────┤                                                                                                                                     
│ text    │ String │ 是   │ 要插入的文本内容               │                                                                                                                                     
└─────────┴────────┴──────┴────────────────────────────────┘

响应示例：                                                                                                                                                                                       
{               
"success": true,

}

POST /book/bookContentInsertWithParagraphs

┌─────────┬────────┬──────┬──────────────────────────────────────────────────────────────┐

{               
"success": true,

8. 搜索书本内容（RAG向量检索）

POST /book/bookContentSearch

┌─────────────┬─────────┬──────┬─────────────────────────────┐                                                                                                                                   
│    字段     │  类型   │ 必填 │            说明             │                                                                                                                                   
├─────────────┼─────────┼──────┼─────────────────────────────┤                                                                                                                                   
│ book_id     │ String  │ 是   │ 书本ID（搜索范围限定）      │
├─────────────┼─────────┼──────┼─────────────────────────────┤                                                                                                                                   
│ text        │ String  │ 是   │ 搜索文本/查询内容           │                                                                                                                                   
├─────────────┼─────────┼──────┼─────────────────────────────┤                                                                                                                                   
│ limit       │ Integer │ 否   │ 返回结果数量上限，默认 1000 │                                                                                                                                   
├─────────────┼─────────┼──────┼─────────────────────────────┤                                                                                                                                   
│ minDistance │ Integer │ 否   │ 最小距离阈值，默认 0        │
└─────────────┴─────────┴──────┴─────────────────────────────┘

响应示例：                                                                                                                                                                                       
{               
"success": true,
"data": { ... }
}

▎ data 字段为 RAG 服务返回的原始搜索结果。
